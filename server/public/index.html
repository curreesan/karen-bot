👮 Karen Moderation API

v1.0.0
Auth
GET /api/auth/login Redirect to Discord OAuth
GET /api/auth/callback Discord OAuth callback, issues JWT
Moderation Logs
POST /api/logs Save a moderation action from the bot
GET /api/logs Get all moderation logs
GET /api/logs/offenses Get offense counts per user
Health
GET /health Server status check
Discord bot uses a locally-run LLM (Ollama / Llama 3.2) for AI moderation. Dashboard: karen-bot-client.vercel.app
